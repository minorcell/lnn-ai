# lln-ai

## Project Overview

An AI agent designed to create intelligent, interactive applications that adapt to your needs, from chatbots to automated workflows. Built with Next.js and integrated with Langchain, this project provides a powerful foundation for developing adaptive AI solutions with a file-based learning approach.

## Getting Started

To get started with the development server, run one of the following commands:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

## Features

- Utilizes Next.js for server-side rendering and static site generation.
- Integrates Langchain for enhanced file-based learning capabilities.
- Automatically optimizes and loads the Geist font family.
